# Vehicle Comment Automation

Automation script developed in Python to process Excel files with vehicle information, generate dynamic comments based on predefined templates, insert records into SQL Server, create backup reports, and clean processed rows automatically.

## Features

- Read and process Excel files
- Generate dynamic comments using templates
- Insert comments into SQL Server database
- Create Excel backup reports
- Generate execution logs
- Automatically clean processed rows
- Modular and maintainable project structure

## Technologies

- Python
- pandas
- openpyxl
- pyodbc
- SQL Server

## Installation

```bash
pip install -r requirements.txt
```

---

## Run project

```bash
python main.py
```
